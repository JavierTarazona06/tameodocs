# TameoDocs

Les PDF compilés automatiquement sont publiés sur GitHub Pages :

[Ouvrir le site des documents TAMEO](https://javiertarazona06.github.io/tameodocs/)

Le site contient quatre boutons vers les PDF générés : dossier de réalisation, rapport final, template TAMEO et scénario vidéo. Les liens des PDF incluent une version générée par GitHub Actions afin d'éviter l'affichage d'anciennes versions conservées par le cache du navigateur.
